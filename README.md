# Weather Application 🌦️

A responsive weather application built using React that fetches real-time weather data based on user input.

## 🚀 Features
- Search weather by city name
- Displays temperature and weather condition
- Real-time API integration
- Responsive UI design
- Error handling for invalid city names

## 🛠 Tech Stack
- ReactJS
- JavaScript
- CSS
- Weather API (OpenWeather or similar)

## ⚙️ Setup
1. Clone the repo
2. Install dependencies:
   npm install
3. Add your Weather API key in App.js 
4. Run:
   npm start
